MODIFIER_PROPERTY_INCOMING_PHYSICAL_DAMAGE_CONSTANT = 61
This property controls 'physical' shield, if defined both on client and server. Return value on client should be current shield health, as a positive integer, on server - amount of damage blocked. Method Name: `GetModifierIncomingPhysicalDamageConstant`.
