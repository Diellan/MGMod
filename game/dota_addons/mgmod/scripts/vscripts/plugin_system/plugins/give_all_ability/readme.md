# Give All Ability
Simple plugin that gives X ability to Y units when they spawn.

## Options
Target:
* Heroes
* Illusions
* Lane Creeps
* Towers
* Neutrals

Choises:
* Custom (use text box)
* Windwalk
* Blink
* Proxy mine
* Mirror Image
* Finger of Death
* Essence Shift
* Prospector Aura
* Rally
* Raise Dead
